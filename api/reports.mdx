## GET /api/reports/productivity

Generate productivity and performance reports with flexible filtering.

### Query Parameters

<ParamField query="from" type="string">
  Start date for report period (ISO 8601 format)
</ParamField>

<ParamField query="to" type="string">
  End date for report period (ISO 8601 format)
</ParamField>

<ParamField query="clientId" type="string">
  Filter by specific client
</ParamField>

<ParamField query="operatorId" type="string">
  Filter by specific operator (user ID)
</ParamField>

<ParamField query="status" type="string">
  Filter by work order status
</ParamField>

### Response

Returns detailed productivity metrics and statistics.

<ResponseField name="period" type="object">
  Report period information
  
  <Expandable title="Period details">
    <ResponseField name="from" type="string">Start date</ResponseField>
    <ResponseField name="to" type="string">End date</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="summary" type="object">
  Summary statistics
  
  <Expandable title="Summary metrics">
    <ResponseField name="totalOrders" type="number">Total work orders</ResponseField>
    <ResponseField name="completedOrders" type="number">Completed orders</ResponseField>
    <ResponseField name="totalEstimatedHours" type="number">Total estimated hours</ResponseField>
    <ResponseField name="totalActualHours" type="number">Total actual hours worked</ResponseField>
    <ResponseField name="efficiency" type="number">Efficiency percentage</ResponseField>
    <ResponseField name="onTimeDelivery" type="number">On-time delivery rate</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="byClient" type="array">
  Breakdown by client
  
  <Expandable title="Client breakdown">
    <ResponseField name="clientId" type="string">Client ID</ResponseField>
    <ResponseField name="clientName" type="string">Client name</ResponseField>
    <ResponseField name="orderCount" type="number">Number of orders</ResponseField>
    <ResponseField name="totalCost" type="number">Total cost</ResponseField>
    <ResponseField name="completionRate" type="number">Completion rate percentage</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="byOperator" type="array">
  Breakdown by operator
  
  <Expandable title="Operator breakdown">
    <ResponseField name="userId" type="string">User ID</ResponseField>
    <ResponseField name="fullName" type="string">Operator name</ResponseField>
    <ResponseField name="orderCount" type="number">Orders worked</ResponseField>
    <ResponseField name="hoursWorked" type="number">Total hours</ResponseField>
    <ResponseField name="efficiency" type="number">Efficiency percentage</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="byStatus" type="object">
  Count by work order status
</ResponseField>

## Use Cases

### Executive Dashboard

Display real-time KPIs on executive dashboard:
```javascript
const dashboard = await fetch('api/reports/dashboard', {
  headers: { Authorization: 'Bearer ' + token }
}).then(r => r.json());

// Display work order progress
const completionRate = 
  (dashboard.workOrders.completed / dashboard.workOrders.total) * 100;

console.log(`Work Order Completion: ${completionRate.toFixed(1)}%`);
console.log(`Resource Utilization: ${dashboard.resources.busy}/${dashboard.resources.total}`);
```

### Monthly Performance Report

Generate monthly productivity reports:
```javascript
const firstDay = new Date(2026, 2, 1); // March 1
const lastDay = new Date(2026, 2, 31); // March 31

const report = await fetch(
  `api/reports/productivity?from=${firstDay.toISOString()}&to=${lastDay.toISOString()}`,
  { headers: { Authorization: 'Bearer ' + token } }
).then(r => r.json());

console.log(`Efficiency: ${report.summary.efficiency}%`);
console.log(`On-time Delivery: ${report.summary.onTimeDelivery}%`);
```

### Client Performance Analysis

Analyze performance for specific clients:
```javascript
const clientReport = await fetch(
  `api/reports/productivity?clientId=${clientId}&from=2026-01-01`,
  { headers: { Authorization: 'Bearer ' + token } }
).then(r => r.json());

const clientData = clientReport.byClient[0];
console.log(`Orders: ${clientData.orderCount}`);
console.log(`Total Revenue: $${clientData.totalCost}`);
console.log(`Completion Rate: ${clientData.completionRate}%`);
```

### Operator Performance Tracking

Track individual operator productivity:
```javascript
const operatorReport = await fetch(
  `api/reports/productivity?operatorId=${operatorId}`,
  { headers: { Authorization: 'Bearer ' + token } }
).then(r => r.json());

const operator = operatorReport.byOperator[0];
console.log(`${operator.fullName}:`);
console.log(`  Orders Completed: ${operator.orderCount}`);
console.log(`  Hours Worked: ${operator.hoursWorked}`);
console.log(`  Efficiency: ${operator.efficiency}%`);
```

## Best Practices

1. **Use date ranges**: Always specify date ranges for large datasets to optimize query performance
2. **Cache dashboard data**: Dashboard metrics can be cached client-side for 1-5 minutes to reduce API calls
3. **Schedule reports**: Generate productivity reports on a schedule (daily, weekly, monthly) for consistent tracking
4. **Compare periods**: Fetch reports for multiple periods to identify trends
5. **Export data**: Use report data for Excel/CSV exports or custom visualizations
6. **Monitor KPIs**: Set alerts based on efficiency, on-time delivery, or other critical metrics

## Metrics Definitions

### Efficiency

Calculated as: `(Estimated Hours / Actual Hours) * 100`
- Values > 100% indicate work completed faster than estimated
- Values < 100% indicate work took longer than estimated

### On-Time Delivery Rate

Percentage of orders delivered by or before commitment date:
`(Orders Delivered On Time / Total Delivered Orders) * 100`

### Completion Rate

Percentage of orders in completed or delivered status:
`(Completed + Delivered Orders / Total Orders) * 100`

### Resource Utilization

Percentage of resources actively working:
`(Busy Resources / Total Active Resources) * 100`

# Welcome to DRAIT Mini-MES

DRAIT Mini-MES is a web-based Manufacturing Execution System designed specifically for metalworking and smithy shops. It provides complete traceability, work order management, resource tracking, and real-time production monitoring in a modular, easy-to-deploy package.

## What is DRAIT Mini-MES?

DRAIT Mini-MES bridges the gap between shop floor operations and business management. It tracks every step of your production process - from client quotations to work order completion - providing visibility into resource utilization, material consumption, and production status in real-time.

### Built for Metalworking Shops

Whether you're running a small smithy or a larger metalworking operation, DRAIT Mini-MES adapts to your workflow:

- **Client Management** - Track clients, generate quotations, and convert approved quotes to work orders
- **Production Planning** - Create and assign work orders to personnel and machinery
- **Real-Time Tracking** - Monitor work order progress with status updates and event logging
- **Resource Management** - Manage personnel, machinery, and equipment availability
- **Material Control** - Track inventory, consumption, and stock adjustments
- **Traceability** - Complete audit trail of all operations and changes

## Technology Stack

DRAIT Mini-MES is built with modern, production-ready technologies:

- **Frontend**: React 18 + TypeScript + Vite + TanStack Query + Recharts
- **Backend**: NestJS + TypeScript + Prisma ORM + JWT Authentication
- **Database**: PostgreSQL 16
- **Infrastructure**: Docker + Docker Compose + Nginx reverse proxy

## Who Should Use This?

DRAIT Mini-MES is designed for:

- **Shop Owners** - Get visibility into operations, monitor KPIs, manage users and audit logs
- **Supervisors** - Plan production, assign work orders, track progress, generate reports
- **Operators** - View assigned work orders, log progress, record material consumption
- **Technical Teams** - Deploy, configure, and integrate via REST API
